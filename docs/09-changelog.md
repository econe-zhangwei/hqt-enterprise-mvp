# 09 Changelog - 文档变更日志

## 目的
集中记录文档体系的版本变更，便于追踪“何时、为何、影响了什么”。

## 适用范围
适用于 `docs/00-08` 所有文档变更记录。

## 定义与术语
- 版本：文档体系的阶段标识。
- 影响模块：受变更影响的页面、接口或流程。
- 变更类型：新增、修改、删除。

## 主内容
### 1. 记录规则
1. 任一文档内容变更都必须记录。
2. 同一天多次变更可合并为一次发布记录。
3. 记录应包含“变更原因 + 影响范围”。
